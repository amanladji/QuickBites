# 🍔 QuickBite — Full Stack Food Delivery App

**Stack:** Spring Boot 3 + MongoDB Atlas + Vanilla JS  
**Deploy:** Render (backend web service + frontend static site)

## Live
- Frontend: https://quickbite-frontend.onrender.com
- Backend API: https://quickbite-backend.onrender.com

## Features
- ✅ Signup / Login (JWT Auth)
- ✅ Browse Restaurants & Menu
- ✅ Add to Cart & Place Orders  
- ✅ Coupon / Offer Validation
- ✅ Order Tracking by ID
- ✅ Order History
- ✅ QuickCoins Loyalty System
- ✅ Referral Codes
- ✅ Newsletter Subscription
- ✅ Search Restaurants & Menu Items
